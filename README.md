# Pico CNN

## TL;DR
LeNet-5 implementation as proposed by Yann LeCun <a id="cit_LeCun1998">[[LeCun1998]](#LeCun1998)</a>
```{bash}
mkdir build
cd build
cmake ..
make lenet_tanh
./lenet_tanh PATH_TO_MNIST_DATASET PATH_TO_WEIGHT_FILE
```
