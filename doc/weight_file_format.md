# Weight File Format
## General Structure
```
MAGIC_NUMBER
NAME
NUM_LAYERS
LAYER_0_NAME
LAYER_0_KERNEL_HEIGHT
LAYER_0_KERNEL_WIDTH
LAYER_0_NUM_KERNELS
LAYER_0_KERNEL_0_VALUE_[0,0]
LAYER_0_KERNEL_0_VALUE_[0,1]
LAYER_0_KERNEL_0_VALUE_[0,2]
...
LAYER_0_KERNEL_0_VALUE_[LAYER_0_KERNEL_HEIGHT,LAYER_0_KERNEL_WIDTH]
LAYER_0_KERNEL_1_VALUE_[0,0]
...
LAYER_0_KERNEL_1_VALUE_[LAYER_0_KERNEL_HEIGHT,LAYER_0_KERNEL_WIDTH]
...
...
LAYER_0_KERNEL_{LAYER_0_NUM_KERNELS-1}_VALUE_[LAYER_0_KERNEL_HEIGHT,LAYER_0_KERNEL_WIDTH]
LAYER_0_NUM_BIASSES
LAYER_0_BIAS_0
...
LAYER_0_BIAS_{LAYER_0_NUM_BIASES}
...
...
...
LAYER_{NUM_LAYERS-1}_NAME
LAYER_{NUM_LAYERS-1}_KERNEL_HEIGHT
LAYER_{NUM_LAYERS-1}_KERNEL_WIDTH
LAYER_{NUM_LAYERS-1}_NUM_KERNELS
LAYER_{NUM_LAYERS-1}_KERNEL_0_VALUE_[0,0]
...
LAYER_{NUM_LAYERS-1}_NUM_BIASSES
...
LAYER_{NUM_LAYERS-1}_BIAS_{LAYER_{NUM_LAYERS-1}_NUM_BIASES}
```
